# A ggplot theme for aesthetic plotting of buzzdetect results.

A ggplot theme for aesthetic plotting of buzzdetect results.

## Usage

``` r
theme_buzzr(base_size = 10, mode = "light")
```

## Arguments

- base_size:

  What should the size of the text elements on the plot be?

- mode:

  Should the theme take on a light or a dark color palette?

## Value

A ggplot2 theme
